# File Management Frontend

Frontend cho hệ thống quản lý file với React, TypeScript, và Ant Design.

## Cài đặt

```bash
yarn install
```

## Chạy ứng dụng

```bash
# Development
yarn dev

# Build
yarn build

# Preview
yarn preview
```

## Tính năng

- Đăng nhập với JWT
- Quản lý users (Admin)
- Quản lý projects và files (User)
- Upload/download files
- Phân quyền và chia sẻ files
- Lịch sử file versions
